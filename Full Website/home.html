Conor's Apple Blog

The purpose of this website is to blog about my favourite apples

About

Ever since I was a kid, I have always enjoyed apples. There are too many types to count, so I've created this blog to describe a few of my favourites.

What is an apple?

An apple is a roughly spherical fruit. Taste can vary from sweet to sour.

More Information (https://en.wikipedia.org/wiki/Apple)

Rating Method

The rating system I use gives each apple an overall score out of 15 points. This score is calculated based on the apple's crunch, flavour, and size
